Working with Inline Block

Let's say you have a group of objects that are similiar in size and shape. A common scenaria would be if you has multiple images or icons that you are setting by side in a row. In our case, we'll use boxes. The easiest way set-up is to set them in parent container with text align:center, to center them and then gice each element a display:inline or inline-block. FYI, you have to declare a width in order for this work. Just keeping it real.
